# 漂流瓶 v2

基于 Supabase 的漂流瓶应用

## 功能

- 🍾 **扔瓶子** - 写下你的心声，投放大海
- 🤲 **捞瓶子** - 随机捞取其他用户的漂流瓶
- 💬 **评论** - 对瓶子进行评论互动
- 💌 **私信** - 和瓶主或评论者私聊
- 📱 **移动端优化** - 完美适配手机
